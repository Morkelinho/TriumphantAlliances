Kids tech classes

Where curiosity turns into code

Hands-on classes for young learners — from first blocks of code to real robots, real websites, and real data. Hover any class for details.

Ages 6+

Scratch Block Coding

An introduction to storytelling, sequencing, and algorithmic thinking. Kids build their own animated stories and games using visual, drag-and-drop blocks — no typing required to start thinking like a programmer.

Ages 9+

MakeCode

Building robots and IoT designs. Learners code physical devices and watch their logic move, light up, and respond in the real world — the moment code stops being abstract.

Ages 12+

Web Design

HTML, CSS, and JavaScript, built and shipped for real — from CodeSandbox through to a live site on Netlify. Every learner walks away with something actually published on the internet.

Ages 13+

Analytics

Data structuring through Excel and Power BI. Turning spreadsheets into dashboards, and raw numbers into decisions — the same skills used in the working world, taught early.

Grade 8 – Grade 12

Computer Applied Technology (CAT)

Curriculum-aligned CAT support from Grade 8 through to matric, built around the exact syllabus learners are being assessed on — practical, exam-focused, and paced to the school term.

Classes run in small groups, matched to age and experience level.
